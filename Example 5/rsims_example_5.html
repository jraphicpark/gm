Regular Expression
From the given list, write a regular expression that would return names in the format of 'last name, first name', but omit names that begin with the letter C. Provide the search expression as well as the replace expression (i.e. /^(.*)$/i => $1)

RegEx for alternating names: /(\w+)\s(\w+)/
